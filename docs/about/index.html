HouneTeam

Our Story

HouneTeam is a small game studio based in Japan. We make idle games: resources accumulate, upgrades compound, and the game keeps running when you close it.

Our first game is Lumo Idle Park, an idle park builder on Android. Trees produce resources on timers, boxes fill and move to the warehouse automatically, upgrades unlock new zones. One published title, more in development.

Small team. We track what works and fix what doesn't. Analytics in from day one.

How We Work

Three things we actually do.

1

Mobile Only

Touch controls, short session loops, no mouse required. Everything is built for phones.

2

Designed for Retention

We track where players stop coming back and fix it. First-session fun is table stakes. The system has to hold at day 30.

3

Tested Before Ship

Tested on real devices. No placeholder graphics ship. Crashes get fixed before updates go out.
