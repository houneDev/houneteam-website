HouneTeam

Solo indie developer based in Japan. Casual mobile games with Y2K soul.

Our Story

HouneTeam is a one-person indie studio based in Japan. I make casual mobile games with a Y2K aesthetic: bold colors, tactile interfaces, and a feel that is distinctly not corporate.

First game is Lumo Idle Park, an idle park builder on Android. Resources accumulate while you are away, upgrades unlock new zones, and the game keeps running when you close it. More titles in development.

No publisher. Full pipeline from concept to store listing. Analytics from day one, tested on real devices before every release.

How We Work

Three things we actually do.

1

Y2K Aesthetic

Bold colors, warm gradients, satisfying taps. Games that look opinionated, because they are.

2

Character Over Content

Small worlds with their own feel. Not story-heavy, but never hollow. Every element earns its place.

3

Designed for Retention

First session fun is a baseline. The system has to hold at day 30. Sessions are short, but players come back.
